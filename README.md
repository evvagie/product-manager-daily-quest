# Yuno

Yuno is an interactive online simulation exploring decision-making from a product management perspective through immersive scenarios and branching choices.

This project was built as a learning-focused, exploratory project during a vibe-coding bootcamp organised by IQ Project (June 2025).

## Tech stack
- Vite  
- TypeScript  
- React  
- shadcn/ui  
- Tailwind CSS  

## My role
End-to-end design and development, including concept definition, scenario logic, and implementation.

Exploratory / non-production project.
